# Monitoring Plan

## Data Drift

Monitor changes in customer behaviour metrics such as recency, frequency, monetary value, sessions, and campaign engagement.

## Prediction Distribution

Track the percentage of customers classified as high, medium, and low risk.

## Business Outcomes

Monitor:

* churn rate
* retention rate
* campaign response rate

## API Reliability

Track:

* response time
* error rate
* failed requests

## Retraining Triggers

Retrain the model when:

* model performance declines
* customer behaviour changes significantly
* data drift exceeds acceptable limits
